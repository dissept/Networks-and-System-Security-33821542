# Weekly Summaries
-----------------------
# ---- Week 02: RSA & Secure Sockets ----
# - RSA key generation using Python's cryptography library
# - AES+RSA hybrid encryption workflow
# - Secure message exchange via socket programming
# - Reflection on asymmetric crypto and secure key handling

# ---- Week 03: Authentication & Access Control ----
# - Password strength analyser with entropy scoring
# - Hashing comparison: MD5 / SHA256 vs bcrypt
# - Salt + pepper demo & TOTP 2FA integration
# - Built a full authentication system class
# - Reflection on modern authentication security

# ---- Week 04: Malicious Software ----
# - File integrity checker (SHA-256 baselines)
# - Change detection for tampered files
# - Signature-based malware scanner (pattern matching)
# - Worm propagation simulator
# - Designed layered countermeasures and defences
# - Reflection on malware behaviour and detection trade-offs

# ---- Week 05: Web Security (Wapiti) ----
# - Automated scanning using Wapiti
# - Analysis of CSP issues, cookie flags, missing headers, XSS
# - HTML report viewer embedded in notebook
# - Reflection on automated scanners vs manual testing

# ---- Week 06: Binary Analysis & Static Triage ----
# - Computed MD5, SHA1, SHA256 hashes (IOCs)
# - Extracted strings from PE binaries
# - Parsed PE headers + import tables using pefile
# - YARA-based signature matching
# - Static triage workflow for SOC-style analysis
# - Reflection on static vs dynamic analysis

# ---- Week 07: Penetration Testing ----
# - Passive reconnaissance (IP, WHOIS-like lookup)
# - Black-box HTTP header enumeration
# - Python port scanner + python-nmap automation
# - Reflection on ethics, scope and realistic recon workflows

# ---- Week 08: Portfolio Feedback ----
# - Reflection on documentation quality and professional presentation
# - Improvements based on marking criteria

# ---- Week 09: GenAI Security ----
# - Deployed local LLMs using Ollama
# - Tested prompt injection, data poisoning drift, model inversion
# - Observed model extraction patterns
# - Designed mitigation strategies (governance, filtering, monitoring)
# - Reflection on GenAI attack surfaces
